Stream has just one method to be implemented, but it has large attached libraries for:

  • building stream pipeline
  • using stream data

By Scalqa convention, method names, which return Stream, are ended with 'Stream'

"ABCD".charStream.tp  // Prints Stream(A, B, C, D)
